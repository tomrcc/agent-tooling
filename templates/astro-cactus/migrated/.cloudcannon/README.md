# Welcome to Astro Cactus

This is your content management dashboard. Here's a quick guide to managing your site.

## Quick links

- [Blog posts](cloudcannon:collections/post)
- [Notes](cloudcannon:collections/note)
- [Tags](cloudcannon:collections/tag)
- [Pages](cloudcannon:collections/pages)

## Blog posts

Blog posts are your main content. Each post has a title, description, publish date, and optional tags, cover image, and pinned status.

To create a new post, open the [Blog posts](cloudcannon:collections/post) collection and click **+ Add**. Posts are written in Markdown using the content editor.

**Pinned posts** appear at the top of the homepage. Toggle the "Pinned" switch in the sidebar to pin a post.

**Draft posts** are hidden from the live site. Use the "Draft" toggle to control visibility.

**Tags** help organize posts. You can pick from existing tags or create new ones.

**Cover images** appear as hero images at the top of a post. Add one via the sidebar.

## Notes

Notes are shorter, more casual posts. Create them from the [Notes](cloudcannon:collections/note) collection.

## Tags

[Tags](cloudcannon:collections/tag) let you add a custom title, description, and introductory text to tag pages. Tag pages are automatically created whenever a blog post uses a tag — the tag collection entry adds optional extra context.

## Pages

The [Pages](cloudcannon:collections/pages) collection contains the homepage and about page. Open them to edit text directly in the visual editor.

## Content editing tips

- Use the **visual editor** to edit pages and see changes in real time
- Use the **content editor** for writing blog posts and notes in Markdown
- Use the **data editor** (sidebar) to update post metadata like tags, dates, and images
- Some Markdown features (admonition blocks, GitHub repo cards) are only visible on the live site after building — they won't appear in the editor preview

## Search

The site includes built-in search powered by Pagefind. The search index updates automatically whenever the site builds.
