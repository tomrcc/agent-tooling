# Welcome to Astro Nano

Astro Nano is a minimal portfolio and blog. This guide will help you manage your content.

## Quick links

- [Blog posts](cloudcannon:collections/blog) — articles and updates
- [Projects](cloudcannon:collections/projects) — portfolio items with optional demo and repo links
- [Work experience](cloudcannon:collections/work) — employment history
- [Pages](cloudcannon:collections/pages) — the homepage

## Blog posts

Each blog post has a **title**, **description**, **date**, and optional **draft** toggle. Draft posts are hidden from the live site.

To create a new post, open the Blog collection and click **+ Add**. Write your content using the editor and publish when ready.

## Projects

Projects are similar to blog posts but also include optional **Demo URL** and **Repo URL** fields for linking to live demos and source code.

## Work experience

Work entries appear on the Work page and the homepage. Each entry has a **company**, **role**, **start date**, and **end date**. Set the end date to "Current" for active positions. The body text describes what you did in the role.

## Pages

The homepage can be edited using the visual editor. Click on text elements directly to make changes.

## Site settings

Site-wide settings like the site name, email, social links, and navigation are managed by a developer in the project code. Contact your developer to update these.
